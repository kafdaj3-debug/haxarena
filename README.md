# GameHubArena

HaxArena - Oyun platformu projesi

## Teknolojiler

- Frontend: React + TypeScript + Vite
- Backend: Express + TypeScript
- Database: PostgreSQL (Neon.tech)
- Deployment: Render.com (Backend) + Netlify (Frontend)

## Kurulum

```bash
npm install
npm run dev
```

## Deployment

Backend: Render.com
Frontend: Netlify
